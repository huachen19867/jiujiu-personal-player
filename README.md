# 99新自用唱机

手机端本地音乐播放器。首版目标是替代臃肿在线音乐 App 的本地播放主线：用户在手机浏览器里选择本地音频文件或导入文件夹，应用生成临时播放列表并播放。

## 快速进入

在 `C:\AI\设计\自用99新唱机` 里双击 `start-player.cmd`，它会启动本地开发服务并打开 `http://127.0.0.1:5173/`。

## 首版范围

- 手机端 PWA，优先适配窄屏触控。
- 支持手动多选音频文件。
- 支持导入文件夹；浏览器不支持目录选择时回退到多选文件。
- 支持播放、暂停、上一首、下一首、进度拖动、音量、顺序/循环/随机模式。
- 支持当前播放高亮、移除歌曲、清空列表。
- 播放列表元数据可保存到浏览器本地；音频文件本体不上传、不长期保存。
